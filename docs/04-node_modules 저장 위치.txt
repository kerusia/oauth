[질문] node_modules는 Bind Mount가 좋을까? Named Volume이 좋을까?

[확인]
1. Bind Mount
    Nodejs 컨테이너에 들어가서 npm install 해주고 git 예외 등록만 해주면 손쉽게 사용 가능하다.
2. Named Volume
    처음 컨테이너가 기동될 때 root 권한으로 생성되므로 권한 변경이 필요하다.
        (1) 첫 기동시 변경 하는 방법
            docker compose run --rm --user root oauth-nodejs chown -R 1000:1000 /app/node_modules
        (2) sleep infinity 실행 후 컨테이너에 직접 들어가서 변경하는 방법
            docker exec -it --user root oauth-nodejs bash
            chown -R node:node /app/node_modules
    그리고 npm install을 해주면 그후로 오류 없이 사용가능하다.
