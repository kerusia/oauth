[질문] JPA 설정하는 방법이 무엇인가?

[확인]
1. MariaDB 에서 사용자와 데이터베이스를 생성한다.
2. MariaDB 클라이언트 의존성을 추가한다.
    - runtimeOnly org.mariadb.jdbc:mariadb-java-client:3.3.3
3. DataSource 환경정보를 설정한다.
    [application.yml]
    spring:
      datasource:
        driver-class-name: org.mariadb.jdbc.Driver
        username: oauth-app
        password: oauth-app
        url: jdbc:mariadb://localhost:3306/oauth_db?useSSL=false

4. JPA 의존성을 추가한다.
    - implementation org.springframework.boot:spring-boot-starter-data-jpa
5. JPA 환경정보를 설정한다.
    [application.yml]
    spring:
      jpa:
        database: mysql
        database-platform: org.hibernate.dialect.MySQL8Dialect
        generate-ddl: true
        hibernate:
          ddl-auto: create
        show-sql: true
